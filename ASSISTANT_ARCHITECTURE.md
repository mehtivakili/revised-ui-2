# معماری دستیار آفلاین همیار دوربین

دستیار در `/assistant` و به‌صورت ویجت شناور در همه صفحات در دسترس است. تمام پردازش در مرورگر کاربر انجام می‌شود: بدون API، بدون کلید و توکن، و بدون ارسال پیام به هیچ سرویس بیرونی.

## زنجیره پردازش

```
پیام فارسی
  → normalizePersian    (تاشدن ی/ک عربی، حذف اعراب، ZWNJ، ارقام فارسی)
  → digitizeNumberWords (تبدیل «بیست و چهار» و «۵۰ میلیون» به عدد)
  → checkDomain         (نگهبان خارج‌از‌حوزه)
  → vectorize           (هش‌سازی: تک‌واژه + دوتایی + سه‌نویسه‌ای)
  → DeepClassifier      (شبکه عصبی → توزیع احتمال روی ۴۲ نیت)
  → extractSlots        (استخراج عدد و واحد)
  → skill | knowledge | catalog
  → پاسخ فارسی + فرض‌ها + پیوند به ابزار متناظر
```

## شبکه عصبی

| مشخصه | مقدار |
| --- | --- |
| معماری | پرسپترون چندلایه: ۱۰۲۴ → ۱۹۲ → ۹۶ → ۴۲ |
| فعال‌سازی | ReLU در لایه‌های پنهان، Softmax در خروجی |
| منظم‌سازی | Dropout معکوس ۰.۱۵ و Weight Decay |
| بهینه‌ساز | Adam با نرخ یادگیری ۰.۰۰۶ |
| تابع هزینه | آنتروپی متقاطع |
| آموزش | ۹۰ دوره، Mini-batch ۳۲ |
| دقت روی پیکره | ۱۰۰٪ |

لایه اول ورودی خلوت (Sparse) می‌خواند؛ هزینه آن با تعداد ویژگی فعال هر جمله (حدود ۷۰) رشد می‌کند نه با کل فضای هش. آموزش روی نخ اصلی اما زمان‌بندی‌شده در برش‌های ۴۰ میلی‌ثانیه‌ای انجام می‌شود تا رابط کاربری قفل نشود، و از `setTimeout` استفاده می‌کند نه `requestAnimationFrame` — چون مرورگر در تب پس‌زمینه rAF را متوقف می‌کند و آموزش می‌خوابید.

وزن‌ها پس از آموزش با کوانتیزه‌سازی int8 در `localStorage` ذخیره می‌شوند (حدود ۲۸۶ کیلوبایت). بازدیدهای بعدی زیر یک ثانیه آماده می‌شوند. کلید ذخیره‌سازی به اثر انگشت پیکره آموزشی گره خورده است، پس تغییر پیکره به‌طور خودکار مدل را بازآموزی می‌کند.

## بازیابی و نگهبان حوزه

- **بازیابی TF-IDF** روی پایگاه دانش، با شباهت کسینوسی به‌علاوه هم‌پوشانی سه‌نویسه‌ای برای تحمل غلط املایی. تا وقتی شبکه در حال آموزش است، پاسخ‌ها از همین لایه می‌آید.
- **نگهبان خارج‌از‌حوزه**: Softmax همیشه چیزی برمی‌گرداند. پیش از اعتماد به شبکه بررسی می‌شود که پیام دست‌کم یک واژه محتوایی مشترک با حوزه داشته باشد. واژگان «قوی» از عنوان و کلیدواژه مقالات و پیام‌های گفت‌وگویی ساخته می‌شود و یکی از آن‌ها کافی است؛ واژگان «ضعیف» (افعال عمومی مانند «میخوام») حداقل دو مورد لازم دارد. متن مقالات عمداً وارد واژگان نمی‌شود، چون نثر عادی فارسی نگهبان را بی‌اثر می‌کند.

## منابع پاسخ

| منبع | محتوا |
| --- | --- |
| `calculation` | همان موتورهای `src/lib/calculators/*` که ابزارهای برنامه استفاده می‌کنند؛ هر عدد در ابزار متناظر قابل بازتولید است |
| `knowledge` | پایگاه دانش فنی فارسی بر پایه EN 62676-4، IEC 60529/62262، IEEE 802.3af/at/bt و پروفایل‌های ONVIF |
| `catalog` | آینه محلی کاتالوگ ddcpersia از طریق `GET /api/catalog/source` |
| `system` | گفت‌وگو، راهنما و پاسخ خارج‌از‌حوزه |

هر پاسخ محاسباتی فرض‌های استفاده‌شده را صریح اعلام می‌کند. قیمت هرگز حدس زده نمی‌شود: اگر محصولی در Snapshot محلی نباشد یا قیمتش ثبت نشده باشد، همان گفته می‌شود.
